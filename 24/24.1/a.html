猜测:满足要求的素数都满足模3余1(除了3)
证明:设 $$ a = 3n+i \quad n \in Z, -1 \leq i \leq 1 \\ a = 3h+j \quad h \in Z, -1 \leq j \leq 1 \\ $$ 所以 $$ \begin {split} p\ (mod\ 3) &= a^{2}+ ab + b^{2} \ (mod\ 3)\\ &= (3n+i)^{2}+ (3n+i)(3h+j) + (3h+j)^{2} \ (mod\ 3) \\ &= i^{2}+ ij + j^{2} \ (mod\ 3) \\ \end {split} $$ 然后排列组合一下可能的情况,2*3 = 6种
i j $i^{2}+ ij + j^{2} (mod 3)$
0 0 0
0 1 1
0 -1 1
1 1 0
1 -1 1
-1 -1 0
所以$ p\ \equiv 0 或 1(mod\ 3) $，因为p是素数且$p \neq 3$,所以$p\ \equiv 1(mod\ 3) $
这里只是推导了p一定(mod 3) = 1,但是并没有证明模3余1的素数p,一定满足$p=a^{2}+ab+b^{2}$,暂且放在这里。
